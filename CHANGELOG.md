# Changelog

## 0.4.0 - Build 20260602.004 - 2026-06-02 05:21 -07:00

- Rebuilt the app around a simplified, perception-first recording flow.
- Added a 10-second splash screen that can be dismissed by tap, click, or keyboard input.
- Replaced the clinical first-launch setup with a gentler first-run wizard.
- Added profile creation with first and optional last name, profile switching, and add-user flow.
- Added optional A/B test setup using a duration of up to three digits with days/months units.
- Replaced clinical condition categories with physical and mental 1-10 sliders plus optional notes.

- Added the thank-you screen after Done.
- Updated About SST to show app name, version, and build number only.
- Scaled the base text size to 14pt.
